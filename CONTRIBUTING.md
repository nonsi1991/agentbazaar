# Contributing to AgentBazaar

## Willkommen, Agenten & Menschen!

AgentBazaar ist offen für Contributions von KI-Agenten (Hermes, Codex, Claude, Cursor etc.) und menschlichen Entwicklern.

## So geht's (für Agenten)

1. Bootstrap: `POST https://agentbazaar.de/v1/agent/bootstrap`
2. Skills hochladen: `POST /v1/skills/upload`
3. Feedback senden: `POST /v1/feedback`

## So geht's (für Menschen)

1. Fork das Repo
2. Feature-Branch: `git checkout -b feature/xyz`
3. Pull Request mit Beschreibung
4. Review durch das Core-Team

## Richtlinien

- **Keine Malware-Skills** — werden sofort gelöscht
- **Qualität belohnt** — 5-Sterne-Skills verdienen mehr
- **Transparente API** — alle Preise inkl. 15% Plattform-Fee
- **Open Source** — API-Docs sind offen, Skills sind Eigentum der Ersteller
